## 📚 关于本项目

这是一个个人学习项目，记录我在学习 OpenClaw 源码过程中的理解、分析和实践。

**目标**：
- 深入理解 OpenClaw 的架构设计
- 学习现代 AI 助手系统的实现
- 分享知识，帮助更多人了解这个项目

**适合读者**：
- 对 AI 助手开发感兴趣的开发者
- 想要学习 Node.js/TypeScript 项目架构的人
- 想要为 OpenClaw 贡献代码的人

## 🚀 快速开始

### 在线阅读

**GitHub Pages**: https://zhatongning.github.io/openclaw-learning

### 本地阅读

```bash
# 克隆仓库
git clone https://github.com/zhatongning/openclaw-learning.git
cd openclaw-learning

# 使用 GitBook 本地预览（需要安装 gitbook-cli）
npm install -g gitbook-cli
gitbook install
gitbook serve

# 访问 http://localhost:4000
```

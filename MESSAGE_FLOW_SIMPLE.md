# OpenClaw 消息流程 - 简洁版

## 📊 核心流程（7 步）

```
┌──────────────┐
│  👤 用户      │
│  Telegram    │
│  发送消息     │
└──────┬───────┘
       │ 1️⃣ "帮我截图我的桌面"
       ▼
┌──────────────┐
│  📱 Telegram │
│  Extension   │
│  接收 & 解析  │
└──────┬───────┘
       │ 2️⃣ 标准消息格式
       ▼
┌──────────────┐
│  🔌 Gateway  │
│  路由 & 分发  │
└──────┬───────┘
       │ 3️⃣ 路由到 Agent
       ▼
┌──────────────┐
│  🤖 Agent    │
│  LLM 决策    │
│  "需要截图"  │
└──────┬───────┘
       │ 4️⃣ 调用 Node
       ▼
┌──────────────┐
│  💻 Node     │
│  Mac mini    │
│  执行截图     │
└──────┬───────┘
       │ 5️⃣ 返回截图文件
       ▼
┌──────────────┐
│  🤖 Agent    │
│  生成响应     │
└──────┬───────┘
       │ 6️⃣ 发送给用户
       ▼
┌──────────────┐
│  👤 用户      │
│  收到截图     │
│  ✅ 完成     │
└──────────────┘
```

---

## 🎯 关键数据流

### 步骤 1: 用户发送
```
用户输入: "帮我截图我的桌面"
平台: Telegram
时间: 11:03:00
```

### 步骤 2: Extension 处理
```
输入: Telegram Webhook
输出: 标准消息
{
  "userId": "8583079349",
  "content": "帮我截图我的桌面",
  "channel": "telegram"
}
```

### 步骤 3: Gateway 路由
```
决策: 
- 消息类型: chat
- 目标: Agent
- 优先级: normal
```

### 步骤 4: Agent 决策
```
LLM 输入: "帮我截图我的桌面"
LLM 输出: {
  "toolCalls": [{
    "name": "screen_capture",
    "arguments": {"display": "primary"}
  }]
}
```

### 步骤 5: Node 执行
```
命令: screencapture -x /tmp/screenshot.png
结果: {
  "success": true,
  "filePath": "/tmp/screenshot.png",
  "size": "1.2MB"
}
```

### 步骤 6: Agent 响应
```
文本: "好的，我已经帮你截取了桌面截图。"
媒体: [图片文件]
```

### 步骤 7: 用户收到
```
📷 [桌面截图]
好的，我已经帮你截取了桌面截图。
```

## ⚡ 性能数据

| 组件 | 耗时 | 说明 |
|------|------|------|
| Telegram Extension | 100ms | 接收和解析 |
| Gateway | 100ms | 路由和分发 |
| Agent (LLM) | 1-2s | 意图识别和决策 |
| Node | 500ms | 执行截图 |
| **总计** | **3-4s** | 端到端 |

## 💡 快速参考

### 消息流向
```
用户 → Telegram → Extension → Gateway → Agent → Node → Agent → Gateway → Extension → Telegram → 用户
```

### 关键技术
- **Telegram**: grammY 框架
- **Gateway**: WebSocket + Express
- **Agent**: LLM (OpenAI/Anthropic/GLM)
- **Node**: 本地设备 API

### 文件位置
- **截图**: `/tmp/screenshot_[timestamp].png`
- **日志**: `~/.openclaw/logs/`
- **配置**: `~/.openclaw/openclaw.json`

# OpenClaw Subagent 使用指南

> 如何创建和管理子代理执行任务

---

## 📊 什么是 Subagent？

**Subagent** 是 OpenClaw 的子代理系统，可以：
- 🔄 在后台执行长时间任务
- 🎯 处理复杂的编码任务
- 📝 生成文档或代码
- 🔧 自动化工作流程

---

## 🚀 基本使用

### 方式 1：直接在对话中请求

**最简单的方式**：直接告诉我你想做什么，我会自动判断是否需要启动 subagent。

**示例**：
```
帮我写一个 Python 爬虫
用 Claude Code 重构这段代码
创建一个文档说明如何使用 OpenClaw
```

---

### 方式 2：明确指定使用 subagent

**语法**：
```
用 subagent 帮我...
启动一个子代理去...
在后台执行...
```

**示例**：
```
用 subagent 帮我写一个 Python 爬虫，爬取豆瓣电影 Top 250
启动一个子代理去重构这段代码
在后台帮我生成项目文档
```

---

## 🎯 Subagent 类型

### 1. **编码 Subagent**
- 用途：写代码、重构、调试
- 触发词：`写一个...`, `重构...`, `修复...`

**示例**：
```
用编码 subagent 写一个 Telegram Bot
```

---

### 2. **文档 Subagent**
- 用途：生成文档、写说明
- 触发词：`生成文档`, `写 README`

**示例**：
```
生成 API 文档
```

---

### 3. **分析 Subagent**
- 用途：代码分析、性能优化
- 触发词：`分析...`, `优化...`

**示例**：
```
分析这个项目的性能瓶颈
```

---

## 🔧 高级用法

### 指定工作目录

**语法**：
```
在 [目录] 中用 subagent [任务]
```

**示例**：
```
在 /Users/zha/projects/my-app 中用 subagent 添加单元测试
```

---

### 指定模型

**语法**：
```
用 [模型] subagent [任务]
```

**示例**：
```
用 GLM-5 subagent 写一个 Python 脚本
用 Claude subagent 重构代码
```

---

### 后台运行

**语法**：
```
在后台用 subagent [任务]
```

**示例**：
```
在后台用 subagent 爬取 100 个网页
```

**特点**：
- ✅ 不阻塞当前对话
- ✅ 完成后通知你
- ✅ 可以继续其他任务

## 💬 常见问题

### Q: Subagent 会阻塞对话吗？
**A**: 不会！Subagent 在后台运行，你可以继续对话。

### Q: 如何知道 Subagent 完成了？
**A**: 完成后我会主动通知你，并展示结果。

### Q: 可以同时运行多个 Subagent 吗？
**A**: 可以！支持多个 subagent 并行运行。

### Q: Subagent 失败了怎么办？
**A**: 我会告诉你失败原因，你可以让我重试或调整任务。
